# SendMsgApp

Aplicación Android simple que muestra una lista de contactos con fotos en una cuadrícula. Al tocar un contacto, abre WhatsApp con un mensaje predefinido ("Hola! Soy Oscar. Llamame por favor") listo para enviar a ese contacto.

Básicamente es un atajo para enviar un mensaje de WhatsApp rápido a contactos seleccionados.

## Observación
Los números de teléfono y las fotos de los contactos no están incluidos en el repositorio por razones de privacidad. Para usar la aplicación, agregue los números de teléfono correspondientes en el archivo `MainActivity.kt` y añada las imágenes de los contactos en la carpeta `app/src/main/res/drawable/`.
